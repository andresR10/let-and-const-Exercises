var PI = 3.14; PI = 42; let PI = 3.14; PI = 'new PI'; const PI = 3.14; PI.push('15'); // 1. var can be redeclared and reassigned and works with a function scope. let can be reassigned but not redeclared and works with block scope. 2. const cannot be redeclared or reassigned but can be mutated. works with block scope 3. let allows for reassign but const does not 4. hositing is moving declarations to the top of the scope it is defined in.
